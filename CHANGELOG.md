# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.0.0] - 2026-03-29

### Added

- **docs-system** plugin — documentation skill with Diataxis framework, 8 templates (README, quickstart, how-to, API reference, ADR, RFC, migration guide, runbook), tone and voice guide, code examples guide, linting integration (Vale + markdownlint), and a tech-writer-reviewer agent
- **ddd-design** plugin — structured Domain-Driven Design discovery sessions covering strategic (subdomains, bounded contexts, context maps) and tactical (aggregates, entities, value objects, domain events) modeling
- **create-hook** plugin — step-by-step guide for creating Claude Code hooks with shell scripts, including template, stdin JSON reference, and testing instructions
- 1Password SSH agent setup reference guide (macOS and WSL2)
- Claude Code security hardening reference guide (permissions, deny rules, sandbox)
